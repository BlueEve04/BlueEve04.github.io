0 安装

按住不表，在官网自行下载即可

1 终端优化

1.1 ZSH

首先安装ZSH（）

很简单：

sudo apt install zsh

好的我们现在已经有了zsh，下面让我们加上一点调味剂（oh-my-zsh）

1.2 Oh-My-Zsh

Oh My Zsh 是什么

Oh My Zsh 是一款社区驱动的命令行工具，正如它的主页上说的，Oh My Zsh 是一种生活方式。它基于 zsh 命令行，提供了主题配置，插件机制，已经内置的便捷操作。给我们一种全新的方式使用命令行。

Oh My Zsh 是基于 zsh 命令行的一个扩展工具集，提供了丰富的扩展功能。

使用 chsh -s /bin/zsh 命令将 zsh 设置为系统默认 shell。

第一次进zsh会有新手引导，进去之后直接q出来

然后（记得先把wget，git，curl装上：sudo apt install git wget curl）

使用 curl 下载脚本并安装：

#没有梯子的使用国内镜像源
sh -c "$(curl -fsSL https://gitee.com/shmhlsy/oh-my-zsh-install.sh/raw/master/install.sh)"

或者使用 wget 下载脚本并安装：

sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

然后同意使用 Oh-my-zsh 的配置模板覆盖已有的 .zshrc

接下来我们开始配置：

1.3 优化

修改配置文件：nano ./.zshrc

重载配置：source ~/.zshrc

1.3.1主题：powerlevel10k

https://github.com/romkatv/powerlevel10k

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

还是使用 vim 编辑 .zshrc的ZSH_THEME字段：

ZSH_THEME="powerlevel10k/powerlevel10k"

记得下载字体：https://github.com/romkatv/powerlevel10k?tab=readme-ov-file#meslo-nerd-font-patched-for-powerlevel10k

1.3.2插件

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

然后：在zshrc里改成：

plugins=(
     # other plugins...
     zsh-autosuggestions
     zsh-syntax-highlighting
     z
     git
)

4 启动项修改和开机自动挂载

首先 

 sudo apt update
 sudo apt install ntfs-3g

然后找你硬盘的uuid:

sudo lsblk

再创建挂载点

选择一个目录作为挂载点。如果没有现成的目录，你可以创建一个：

sudo mkdir /mnt/c

再编辑：

sudo nano /etc/fstab

加上你的硬盘，像这样：

# nvme0n1p1 (Data)
UUID=C832719F327192E4    /mnt/d    ntfs    defaults    0    2

# nvme1n1p3 (System)
UUID=CF0594CDA5859D1A    /mnt/c    ntfs    defaults    0    2

完！

关于国内Electron npm下载慢速的解决方法

0 前提

我本人之前一直是不喜欢用cnpm（没啥原因），但是现在不知怎么回事完全没法用npm install electron了，遂上网查找

如果你和我一样用npm下载一些依赖和程序失败也可以试试这个

不会只有我才知道吧呜呜呜呜呜

1 方法

先这样，再那样（

首先运行：

npm install -g cnpm --registry=https://registry.npmmirror.com

然后：

cnpm install --save-dev electron

还得是cnpm

这问题我以前也经常遇到但是都最后解决了（）
