关于国内Electron npm下载慢速的解决方法

0 前提

我本人之前一直是不喜欢用cnpm（没啥原因），但是现在不知怎么回事完全没法用npm install electron了，遂上网查找

如果你和我一样用npm下载一些依赖和程序失败也可以试试这个

不会只有我才知道吧呜呜呜呜呜

1 方法

先这样，再那样（

首先运行：

npm install -g cnpm --registry=https://registry.npmmirror.com

然后：

cnpm install --save-dev electron

还得是cnpm

这问题我以前也经常遇到但是都最后解决了（）
